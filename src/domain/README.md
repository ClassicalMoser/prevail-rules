Contains all core business logic and game rules for games of Prevail: Ancient Battles

This includes:

1. Entities: Object structures and their encoded meaning
2. Functions: Algorithms for applying game principles. These may be further broken down.
3. Sample Values: Placeholder units, cards, and rule constants until a permanent database is set up.
4. Validation: Checks for valid entiteis and game states. Always outputs boolean, never throws.

The directory also includes utilities and testing helpers.
